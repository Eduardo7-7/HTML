O que é HTML:

Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

O que são tags:

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

Algumas tags:

  • strong = negrito
  • i = itálico
  • h1-h6 = títulos e subtítulos
  • p = paragráfos
  • blockquote = citação/frase/observação
  • u = sublinhado
  • mark = marcar texto
  • sup = parte de cima da linha (*)
  • ol = lista ordenada (números)
  • ul = não ordenada (bolinhas)
  • li = lista
  • a href = "site" title="(aparece um balãozinho)" target="_Blank"(abre outra aba) target="_self"(abre na mesma aba) > (texto que vai exibir na tela)
  • hr = linha
  • small = texto pequeno

Estrutura básica do HTML:

  1. Primeiro criar uma pasta.
  2. index.html = dentro da pasta no visual studio code
  3. html = inicia (primeira coisa a abrir)
  4. head = nome do site (title)
  5. body = corpo do site(tudo dentro vai aparecer no site feito)

Navegar no próprio site:

  • Criar outra pagina no visual studio code e fazer outra estrututa básica HTML (about.html)
  • a href = "about.html(nome da página nova)"> (nome) pra ir
  • a href = "index.html(nome da primeira página)"> (nome) pra voltar
